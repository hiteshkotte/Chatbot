About BiWi AI Tutor

BiWi AI Tutor is a cutting-edge chatbot designed to assist students in the Bildungswissenschaften (BiWi) course at the University of Leipzig. The AI leverages advanced language models, document embeddings, and retrieval systems to provide precise, context-aware responses.

Key Features

  • Intelligent Chatbot powered by LangChain and LangGraph.
  • Document Retrieval using advanced embedding techniques.
  • Multi-model support, integrating OpenAI GPT-4 and Cohere Rerank.
  • Web interface built with Streamlit for an intuitive user experience.
  • Mentoring capabilities tailored for personalized student guidance.
  • Seamless integration with FastAPI for API-based interactions.

System Architecture

The BiWi AI Tutor consists of several interconnected modules:

  • Document Processor: Converts PDF course materials into searchable embeddings.
  • Embedding Index: Stores semantic embeddings for fast retrieval.
  • LangChain Core: Handles query parsing, retrieval, and model integration.
  • Streamlit Frontend: A web interface for chatbot interactions.
  • FastAPI Backend: Exposes endpoints for API-based integrations.

Above is the high-level architecture of the BiWi AI Tutor system.

Technical Details

BiWi AI Tutor uses an embedding-based retrieval mechanism. The core technical components include:

  • LangChain: Handles conversational flow and tool usage.
  • LangGraph: Visualizes the state of agent execution.
  • Azure Integration: Enables hosting custom models securely.
  • Streamlit Interface: Provides a web-based chat UI.
  • FastAPI: Powers the backend for exposing RESTful endpoints.

Environment Setup

To run the project, ensure the following dependencies are installed: 

Docker, Python 3.12, Streamlit, FastAPI

Use the following commands to set up:

git clone https://github.com/your-repo/biwi-ai-tutor.git
pip install -r requirements.txt
python app.py

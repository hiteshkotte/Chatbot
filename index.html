About BiWi AI Tutor

BiWi AI Tutor is an advanced chatbot designed to assist students in the Bildungswissenschaften (BiWi) course at the University of Leipzig. Leveraging cutting-edge language models and document retrieval systems, it provides precise answers to course-related queries, enhancing the overall learning experience.

Key Features

  • Intelligent Chatbot powered by LangChain and LangGraph.
  • Document Retrieval for context-aware responses.
  • Multi-model support, including OpenAI and Cohere integrations.
  • User-friendly web interface built with Streamlit.
  • Mentoring capabilities to support personalized learning.

How It Works

The chatbot processes course materials, stored as documents, to deliver meaningful responses. Students can interact with BiWi AI Tutor through a Streamlit interface or via API endpoints for custom integration.

Why Choose BiWi AI Tutor?

Designed specifically for students in educational sciences, BiWi AI Tutor goes beyond simple Q&A systems by:

  • Providing real-time, contextually accurate answers.
  • Supporting mentoring roles set by teaching staff.
  • Enabling effortless integration with existing tools through APIs.
